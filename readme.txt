Real-time reassignment and tools
--------------------------------

Dependencies
------------

It is necessary to install JUCE and Steinberg VST SDK.


How to compile
--------------

1) Open project of your choice in Introjucer and set correct paths to JUCE modules 
   and to the VST SDK folder. 

2) Save the project (to apply the changes)!

3) Navigate to Builds subdir. and compile.

4) [Optional] When compiling plugin, copy it to the plugin directory of your 
   host app.
